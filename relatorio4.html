Introdução

Neste experimento foi implementado um sistema de led RGB com uso de potenciômetros.

Objetivos

  • Desenvolver um sistema com um led para alternar cores com um sistema RGB controlado por um potenciômetros Arduino.
  • Garantir que todas as ligações e configurações necessárias do hardware estejam completas.
  • Verificar o funcionamento do semáforo conforme o código C++ desenvolvido.

    • Materiais Utilizados

    • Arduino Uno
    • LEDs
    • Resistores
    • Placa de ensaio (breadboard)
    • Fios de conexão
    • Computador com IDE Arduino instalada
    • Potenciômetros

    int pinRed = 6; int pinGreen = 3; int pinBlue = 5; void setup() { pinMode(pinRed, OUTPUT); pinMode(pinGreen, OUTPUT); pinMode(pinBlue, OUTPUT); } void setup() { pinMode(pinRed, OUTPUT); pinMode(pinGreenm OUTPUT); pinMode(pinBlue, OUTPUT); pinMode(potRed, INPUT); pinMode(potGreen, INPUT); pinMode(potBlue, INPUT); Serial.begin(9600); } void loop() {; int valRed = analogRead(potRed); int valGreen = analogRead(potGreen); int valBlue = analogRead(potBlue); valRed = map(valRed, 0, 1023, 0, 255); valGreen = map(valGreen, 0, 1023, 0, 255); valBlue = map(valBlue, 0, 1023, 0, 255); analogWrite(pinRed, valRed); analogWrite(pinGreen, valGreen); analogWrite(pinBlue, valBlue); delay(1000); Serial.print("Codigo RGB\("); Serial.print(valRed;); Serial.print(','); Serial.print(valGreen); Serial.print(valBlue); Serial.println(')'); }

    Procedimentos

  • Realizar a montagem do circuito na placa de ensaio conforme o diagrama.
  • Verificar todas as conexões dos LEDs e resistores com os pinos correspondentes no Arduino.
  • Carregar o código C no Arduino utilizando a IDE Arduino.
