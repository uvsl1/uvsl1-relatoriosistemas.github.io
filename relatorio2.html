Introdução

Neste experimento foi implementado um sistema para controlar o brilho de um led com um potenciômetro.

Objetivos

  • Desenvolver um led com brilho controlado por um potenciômetro no Arduino.
  • Garantir que todas as ligações e configurações necessárias do hardware estejam completas.
  • Verificar o funcionamento do semáforo conforme o código C++ desenvolvido.

    • Materiais Utilizados

    • Arduino Uno
    • LEDs
    • Resistores
    • Placa de ensaio (breadboard)
    • Fios de conexão
    • Potenciômetro
    • Multímetro

    Código Fonte

    int potenciometro = A0; int led = 3; float valor = 0; void setup() { pinMode(potenciometro, INPUT); pinMode(led, OUTPUT); Serial.begin(9600); } void loop() { valor = map(analogRead(potenciometro),0,1023,0,255); analogWrite(led, valor); Serial.println(valor); delay(10); }
