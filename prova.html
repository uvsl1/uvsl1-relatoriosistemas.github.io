Introdução

Neste experimento foi implementado um sistema de semáforo e cancela utilizando um Arduino. Tendo como objetivo agora com uma maior complexidade e com uso de novos artificios como sensores, fazer um retrabalho em cima da primeira prática realizada agora com novas funcionalidades.

Objetivos

  • Desenvolver um semáforo funcional utilizando Arduino.
  • Garantir que todas as ligações e configurações necessárias do hardware estejam completas.
  • Implementar um sistema de cancela para o semáforo com o uso de sensores.
  • Conciliar tempos de resposta dos sensores, abertura da cancela e mudança de estado do semáforo com utilização do codigo em C++.

    • Materiais Utilizados

    • Arduino Uno
    • Servo Motor
    • Sensor ultrassônico HC-SR04
    • LEDs (vermelho)
    • Resistores
    • Placa de ensaio (breadboard)
    • Fios de conexão
    • Computador com IDE Arduino instalada
    • Acrílicos base, cancela e farol
    • Espaçadores base

    Código Fonte

    // Definição dos pinos dos LEDs const int ledVerde = 10; const int ledAmarelo = 9; const int ledVermelho = 8; // Definição dos pinos do sensor ultrassônico const int trigPin = 12; const int echoPin = 13; // Variáveis para calcular a distância long duracao; int distancia; void setup() { // Configuração dos pinos dos LEDs como saída pinMode(ledVerde, OUTPUT); pinMode(ledAmarelo, OUTPUT); pinMode(ledVermelho, OUTPUT); // Configuração dos pinos do sensor ultrassônico pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); // Inicialização da comunicação serial Serial.begin(9600); } void loop() { // Verifica a presença de um veículo digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); // Calcula a distância com base na duração do sinal duracao = pulseIn(echoPin, HIGH); distancia = duracao * 0.034 / 2; // Exibe a distância no Serial Monitor Serial.print("Distancia: "); Serial.println(distancia); // Se a distância for menor que 20 cm, ativa o semáforo vermelho if (distancia < 20) { digitalWrite(ledVerde, LOW); digitalWrite(ledAmarelo, LOW); digitalWrite(ledVermelho, HIGH); } else { // Ciclo normal do semáforo digitalWrite(ledVerde, HIGH); delay(5000); // 5 segundos no verde digitalWrite(ledVerde, LOW); digitalWrite(ledAmarelo, HIGH); delay(2000); // 2 segundos no amarelo digitalWrite(ledAmarelo, LOW); digitalWrite(ledVermelho, HIGH); delay(5000); // 5 segundos no vermelho digitalWrite(ledVermelho, LOW); } delay(500); // Pequena pausa antes do próximo ciclo }

    Procedimentos

  • Realizar a montagem do circuito na placa de ensaio conforme o diagrama e o ja projetado anteriormente.
  • Verificar conexões dos componentes, resistores, pinos, LEDs e o sensor ultrassônico.
  • Carregar o código C++ no Arduino utilizando a IDE Arduino.
  • Teste de funcionamento e correspondencia com o tempo de resposta baseado na distância recebida através do sensor.

    • Resultados

    O semáforo foi implementado com sucesso, utilizando objetos para o teste de distância do sensor ultrassônico, podemos observar a mudança de estado do semáforo conforme a distância do veículo e a abertura ou não da cancela.

    Discussão

    O experimento demonstrou a importância de seguir corretamente o diagrama de montagem e as configurações de pinos no Arduino. Pequenos erros de conexão podem comprometer o funcionamento do circuito. Além disso, a programação correta dos tempos de troca de sinal é essencial para simular com precisão o comportamento de um semáforo.

    Conclusão

    A prática foi bem-sucedida, cumprindo os objetivos propostos. O semáforo junto ao sensor ultrassônico funcionaram de acordo com o diagrama e o codigo implementado.
