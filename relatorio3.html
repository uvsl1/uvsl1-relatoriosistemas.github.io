Introdução

Neste experimento foi implementado um sistema de sensor ultrassonico para ligar um led

Objetivos

  • Desenvolver um led ligado por sensor ultrassonico.
  • Garantir que todas as ligações e configurações necessárias do hardware estejam completas.
  • Verificar o funcionamento do semáforo conforme o código C++ desenvolvido.

    • Materiais Utilizados

    • Arduino Uno
    • LEDs
    • Resistores
    • Placa de ensaio (breadboard)
    • Fios de conexão
    • Computador com IDE Arduino instalada
    • Sensor ultrassonico

    Código Fonte

    int trigger = 8; int echo = 7; int led = 5; float distancia; float tempo; void setup() { pinMode(trigger, OUTPUT); pinMode(echo, INPUT); Serial.begin(9600); pinMode(led, OUTPUT); } void loop() { digitalWrite(led, HIGH); digitalWrite(trigger, HIGH); delayMicroseconds(2); digitalWrite(trigger, LOW); delayMicroseconds(10); tempo = pulseIn(echo, HIGH); distancia = (0.0344 * tempo) / 2; Serial.print("distancia"); Serial.print(distancia); Serial.println("cm"); if(distancia >= 40){ digitalWrite(led, HIGH); } else{ digitalWrite(led, LOW); } }
