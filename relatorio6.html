Introdução

Neste experimento foi implementado um sistema que faz uma hélice girar quando um sensor captar movimento.

Objetivos

  • Desenvolver um programa que faça que toda vez que um sensor capte movimento, a hélice se mova.
  • Garantir que todas as ligações e configurações necessárias do hardware estejam completas.
  • Verificar o funcionamento do semáforo conforme o código C++ desenvolvido.

    • Materiais Utilizados

    • Arduino Uno
    • Placa de ensaio (breadboard)
    • Fios de conexão
    • Computador com IDE Arduino instalada
    • Sensor de movimento

    Código Fonte

    #include < Servo.h> const int trigPin = A1; const int echoPin = A0; const int servoPin = 9; Servo myServo; void setup() { pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); myServo.attach(servoPin); myServo.write(0); Serial.begin(9600); } void loop() { long duration, distance; digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); duration = pulseIn(echoPin, HIGH); distance = duration * 0.034 / 2; int servoAngle; if (distance <= 50) { servoAngle = 0; } else if (distance >= 230) { servoAngle = 180; } else { servoAngle = map(distance, 50, 230, 0, 180); } myServo.write(servoAngle); Serial.print("distancia: "); Serial.print(distance); Serial.print(" cm, angulo: "); Serial.println(servoAngle); delay(15); }
