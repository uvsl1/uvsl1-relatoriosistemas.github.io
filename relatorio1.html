Introdução

Neste experimento foi implementado um sistema de semáforo utilizando um Arduino. O objetivo principal foi garantir que todas as conexões e configurações do Arduino fossem realizadas corretamente para simular o comportamento de um semáforo de trânsito.

Objetivos

  • Desenvolver um semáforo funcional utilizando Arduino.
  • Garantir que todas as ligações e configurações necessárias do hardware estejam completas.
  • Verificar o funcionamento do semáforo conforme o código C++ desenvolvido.

    • Materiais Utilizados

    • Arduino Uno
    • LEDs (vermelho, amarelo, verde)
    • Resistores
    • Placa de ensaio (breadboard)
    • Fios de conexão
    • Computador com IDE Arduino instalada

    Diagrama do Circuito

    Código Fonte

    // Codigo em C++ para controlar o semáforo com Arduino int red = 13;
    int yellow = 12;
    int green = 11;

    void setup() {
      pinMode(red, OUTPUT);
      pinMode(yellow, OUTPUT);
      pinMode(green, OUTPUT);
    }

    void loop() {
      digitalWrite(green, HIGH);
      delay(5000);
      digitalWrite(green, LOW);

      digitalWrite(yellow, HIGH);
      delay(2000);
      digitalWrite(yellow, LOW);

      digitalWrite(red, HIGH);
      delay(5000);
      digitalWrite(red, LOW);
    }

    Procedimentos

  • Realizar a montagem do circuito na placa de ensaio conforme o diagrama.
  • Verificar todas as conexões dos LEDs e resistores com os pinos correspondentes no Arduino.
  • Carregar o código C no Arduino utilizando a IDE Arduino.
  • Testar o funcionamento do semáforo e ajustar o tempo de mudança de estado conforme necessário.

    • Resultados

    O semáforo foi implementado com sucesso, alternando entre as cores verde, amarelo e vermelho com os intervalos de tempo programados. Todas as conexões do Arduino foram realizadas corretamente, garantindo o funcionamento pleno do sistema.

    Discussão

    O experimento demonstrou a importância de seguir corretamente o diagrama de montagem e as configurações de pinos no Arduino. Pequenos erros de conexão podem comprometer o funcionamento do circuito. Além disso, a programação correta dos tempos de troca de sinal é essencial para simular com precisão o comportamento de um semáforo.

    Conclusão

    A prática foi bem-sucedida, cumprindo os objetivos propostos. O semáforo implementado funciona corretamente e todas as ligações e configurações do Arduino foram realizadas de forma satisfatória.
